A modern web-based platform designed to streamline school administration and enhance digital learning.
This system empowers admins, teachers, and students with tools for managing classes, communication, and online courses.

✨ Features
👨‍💼 Admin Control
- Create and manage students, teachers, and courses.
- Assign teachers to classes and monitor enrollment.
- Send announcements to specific classes or globally.
- View and edit student records in a centralized dashboard.
👩‍🏫 Teacher Portal
- Manage assigned classes and students.
- Post announcements directly to their classes.
- Upload and manage courses with video content.
- Track student engagement and class statistics.
🎓 Student Portal
- Access assigned classes and teachers.
- View announcements from admins and teachers.
- Join e-learning courses and watch intro videos.
- Stay updated with communication boards.
💻 E-Learning
- Browse and enroll in premium courses.
- Watch introductory videos for each subject.
- Courses led by expert instructors across multiple subjects.
- Easy-to-use interface for interactive learning.





🚀 Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js / Supabase (authentication & APIs)
- Database: PostgreSQL (via Supabase)
- Deployment: Vercel
